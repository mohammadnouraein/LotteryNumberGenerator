# Lottery Number Generation

# Installation construction

1. install dependencies
    `npm i`

2. build the application
    `npm run build`
    in build procedure 

3. run server socket component
    `npm run server`

4. run client socket component 
    `npm run client`

5. run http server 
    `npm run http-server`

6. run http client web application 
    `npm run http-client`
